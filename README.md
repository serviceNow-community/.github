Welcome to the **unofficial** [ServiceNow Community](https://www.servicenow.com/community) on GitHub. We are a collective of developers, architects, and innovators dedicated to enhancing the ServiceNow ecosystem through open-source collaboration.

## What we do

- Share reusable scripts, UI components, and Spoke integrations.
- Develop boilerplate applications and utility tools.
- Document best practices for the Now Platform.

We believe in breaking down silos and making the world of work better for everyone through transparent, community-driven code. Join us, fork a repo, and let’s build something great.
